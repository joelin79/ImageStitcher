# ImageStitcher

使用前記得安裝對應的OpenCV、opencv_contrib(兩者版本皆為3.4.2)才能使用SIFT套件喔，版本不同可能會有問題！

```
pip install opencv-python==3.4.2.16
pip install opencv-contrib-python==3.4.2.16
```
